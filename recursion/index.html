Writing Recursive Methods

The Fibonacci sequence is a sequence in which the first 2 terms are given as 0 and 1 respectively, and all other terms are made up of the sum of the prior 2 terms.

The first 7 terms are 0, 1, 1, 2, 3, 5, 8 ...

(1) Create a recursive solution that returns the nth term of the Fibonacci sequence.

(1) A recursive solution, typically applied in a method or a function, is a block of code that references itself, or calls itself. There are 2 parts to a recursive solution:

  • base case, in which the block of code send a return to the caller
  • recursive case, in which the block of code calls itself

(2) This code raises the int parameter, num, to the power of the other int parameter, power. The result is then returned. Effectively, num is multiplied by itself power times.

A recursive solution needs a base case and a recursive case.

If the power has a value of 1, we return the num parameter, as num1 is num. This is the base case.

If the value of power is greater than 1, we can call the method recursively, reducing the value of power and multiplying by num.

public int morePowerToYou(int num, int power) {
  if (power==1) {
    return num;
     }
   return (num*morePowerToYou(num, power-1));
  }
