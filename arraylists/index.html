Sorting Arrays

A list, employees, of instances of class, Employee, has been established by method populateEmployees.

ArrayList<Employee> employees = new ArrayList<Employee>()

// Call populateEmployees
populateEmployees(employees);

The array is manipulated based on employee number. Employee has attribute number, and accessor, getNumber. Method format manipulates the array based on attribute number


public static int format(ArrayList<Employee> arrList) {
   for(int index = 0; index < arrList.size(); index++) {
    int pos = index - 1;
     while (pos >= 0 && arrList.get(pos).getNumber() > arrList.get(index).getNumber()) {
       arrList.set(pos + 1, arrList.get(pos));
       pos = pos - 1;
       }
       arrList.set(pos + 1, arrList.get(index));
    }
    }
