Accessing Arrays

Arrays are accessed with an index or subscript and are zero-based - the first index is at 0.

Arrays have to be newed up (instantiated)

int[] nums = {1,44,55,77,3};
int temp = nums[0];

temp has a value of 1.

The largest viable index is at nums.length - 1.

nums[nums.length-1] is 3.

Any index value less than 0 or greater than or equal to nums.length will result in an array index out of bounds error

  1. nums[3] = 1; nums[nums.length-1] = 11;
  2. nums[2] = 11; nums[7]=11;
Which of the following code snippets fills the 3rd and last elements of the array,
int[] nums = new int[7];
Option 1, because the array starts at index 0, the third element is at nums[2]. The last element should be referenced using the length property of the array, less 1.

Dynamic Changes

Sometimes the content of the array changes as it is being processed, for instance by a loop.

int[] nums = {1,44,55,77,3};
for (int index = nums.length-1;
index > 0; index--) {
   if (nums[index-1] > nums[index])
     nums[index] += 2;
  }
