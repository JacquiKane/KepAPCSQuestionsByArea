(2) Use the length property of arrays to find the element in the last row and last column.

The number of rows in the 2D array is described by

nums.length

The number of columns is described by picking any row, and finding the length,

nums[0].length

Remember that arrays are zero-based. The element in the last row, last column is,

nums[nums.length-1][nums[0].length - 1]

(1) All of the even numbers in the array are printed : 2,4,4,88, 8 The enhanced for loop cycles through each row in sequence, and then each item in that row. If the value, modulus 2, is 0, then the item is printed.

(2) The fourth item printed is 4.The outer for loop processes the rows, and the inner for loop iterates through the columns. When row has a value of 0, col iterates through values of 0,1,2,3. The fourth item printed is at numbers[0][3], which is 4.

(3) Pop> is printed 2 times. The row starts at 1, so the second and third rows are processed in the outer while loop. The inner while loop iterates through each item in the row, only 2 of these values are not greater than 50, or 5, so Pop is printed twice.

This causes an array index out of bounds error, because of the row value, team[3][2]. The valid row values for this 2D array are from 0, first row, to the last row, index represented by the value of the number of rows in the array, less 1. Here, the number of rows in the array is 3, so the highest viable index is 2.

The number of columns in the 2D array is also 3, so the highest value that can be used to represent a column is 2.

(1) The player name attribute, row at index 2 (which is the 3rd row), column 3 (fourth column from left), is printed.

(2) All the player names are printed as the 2D array is traversed by this nested loop. We are told that the array has been populated with valid instances, and that the attributes have been set.

(1) -1 may be returned even if the employee number exists, that is because the loop condition should use the number of rows instead of the number of columns.

employees.length gives the number of rows
