Nested If

How many times are Fizz, Buzz, and Pop printed?

for (int index = 0; index < 11; index++) {
   if (index % 2 == 0) {
    System.out.println("Fizz");
  }
else if (index % 3 == 0) {
    System.out.println("Buzz");
  }
else {
    System.out.println("Pop");
  }

Analysis

(1) Fizz is printed 5 times. Fizz is printed when index has a value of 5,7,9,11,13. Each time through the loop the sentry variable is increased by 2. After the fifth time through the loop body, index has a value of 15. Since 15 > 15 is false, the loop executes 5 times only.

(2) Buzz is printed three times, when index has a value of 0, 2, 4. After the third time through the loop, index has a value of 6, so the loop condition is no longer true and the loop finishes executing. Buzz is printed.

(3) Pop is printed 11 times. The loop iterates for each element of the array, which has 11 elements.

Impossible to determine, as the loops are infinte loops.

In (a), index has a starting value of 4, each time through the loop, 3 is added. The loop executes for index values of 4,7,10,13,16,19... The addition of 3 never reaches a value of 17, the loop condition is always true, so the loop is infinite.

In (b), index is incremented by 1, in the for loop statement, but is then decremented in the body of the loop. index remains at a value of 0, and the loop condition is always true.

(1) 1, and 144. The printed value is the product of first and secondfactors. The smallest value happens when firstFactor and secondFactor are at the lowest values, which is 1. The largest value of firstFactor is 12, and the largest value of secondfactor is 12. Therefore the largest value printed is 144

(2) secondFactor should be reset to 1 right before the inner loop. Otherwise, the print will only be seen the first iteration of the outer loop.

int firstFactor = 1, secondFactor = 1;
while (firstFactor < 13) {
   secondFactor = 1;
  while (secondFactor <= 12) {
    System.out.println(firstFactor*secondFactor);
    secondFactor++;
    }
    firstFactor++;
  }
