(1) (!a && !b) || (!a || b))
Apply De Morgan's Law to the expression,
!((a || b) && (a && !b)) → !(a || b) || !(a && !b)
Apply De Morgan's Law to the individual parts,
!(a || b) || !(a && !b) → (!a && !b) || (!a || b)

(2) A truth table for this may reveal a further simplification

a b !a && b !a || b
false false true true
false true false true
true false false false
true true false true

(2) This is false when b is false and a is true. It is true when both a and b are false, or when a and b are true, also when only b is true.
